# Interactive Blueprint

An interactive learning repository for mastering **Blueprint Visual Scripting** in Unreal Engine.
This project documents practical experiments, small systems, and gameplay mechanics built while learning Blueprint step-by-step.

The goal of this repository is to build a strong foundation in visual scripting and gradually move toward advanced gameplay systems and VR interactions.

---

## 🚀 Project Overview

This repository contains various Blueprint experiments and interactive systems developed while learning Unreal Engine. Each module focuses on understanding a specific concept such as:

* Blueprint fundamentals
* Actor and component interactions
* Event-driven scripting
* Player input systems
* Interactive gameplay mechanics
* VR interaction systems (later stages)

The project follows a structured **day-by-day learning approach**.

---

## 🧠 Learning Goals

* Understand the core logic of Blueprint visual scripting
* Build small interactive gameplay systems
* Learn debugging and Blueprint optimization
* Develop VR-ready interaction mechanics
* Prepare for advanced Unreal Engine development
